# Shopify コネクタ

公式: https://docs.workato.com/en/connectors/shopify.html

## Triggers
| 名前 | 説明 |
|---|---|
| New customer | Shopifyで顧客が作成された時にトリガーされる |
| New order | Shopifyで注文が行われた時にトリガーされる |
| New product | Shopifyに商品が追加された時にトリガーされる |
| New product variant | 商品バリエーションが作成された時にトリガーされる |
| New/updated abandoned checkout | 新規または変更された放棄チェックアウトを検知する |
| New/updated customer | 顧客情報が変更された時にトリガーされる |
| New/updated draft order | 新規または変更された下書き注文を検知する |
| New/updated order | 注文情報が変更された時にトリガーされる |
| New/updated product | 商品詳細が変更された時にトリガーされる |
