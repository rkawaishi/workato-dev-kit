# ServiceNow コネクタ

公式: https://docs.workato.com/en/connectors/servicenow.html
Provider: `servicenow`

## Triggers
| 名前 | 説明 |
|---|---|
| New record (real-time) | 選択テーブルの新規レコード（リアルタイム） |
| Updated record (real-time) | 選択テーブルの更新レコード（リアルタイム） |

## Actions
| 名前 | 説明 |
|---|---|
| Create record | レコード作成 |
| Update record | レコード更新 |
| Get record | レコード取得 |
| Search records | レコード検索 |
| Delete record | レコード削除 |

## 備考
- リアルタイムトリガーは ServiceNow 側の Business Rule 設定が必要
- テーブルベースの操作（任意のテーブルを選択可能）
