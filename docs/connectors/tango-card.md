# Tango Card コネクタ

公式: https://docs.workato.com/en/connectors/tango_card.html

## Triggers

| 名前 | 説明 |
|---|---|
| (なし) | このコネクタにはトリガーがありません |

## Actions

| 名前 | 説明 |
|---|---|
| Create an order | 受信者にメールで送信される注文を作成する |
| Search orders | 処理済み注文の一覧を取得する |
| Resend an order | 処理済み注文を同じまたは別のメールアドレスに再送信する |
| Get account by ID | 現在のアカウント残高などの特定のTango Cardアカウント情報を取得する |
