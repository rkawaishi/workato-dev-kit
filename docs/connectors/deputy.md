# Deputy コネクタ

公式: https://docs.workato.com/en/connectors/deputy.html

## Triggers

| 名前 | 説明 |
|---|---|
| New employee | Deputyで新しい従業員が作成されたときにトリガーする |
| New leave | Deputyで新しい休暇が記録されたときにトリガーする |
| New timesheet | Deputyで新しいタイムシートが提出されたときにトリガーする |

## Actions

| 名前 | 説明 |
|---|---|
| Create employee | Deputyに新しい従業員レコードを作成する |
| Create sales data | Deputyに新しい売上データを記録する |
| Create resource | Deputyに新しいリソースを作成する |
| Create task | Deputyに新しいタスクを作成する |
| Get resource | Deputyからリソースの詳細を取得する |
| Search employees | 条件に一致する従業員レコードを検索する |
| Search resources | 条件に基づいてリソースを検索する |
| Search operational units | Deputyのオペレーショナルユニットを検索する |
| Update employee | 既存の従業員情報を更新する |
| Update resource | リソースの詳細を更新する |
| Associate employee | 従業員を組織ユニットに関連付ける |
| Unassociate employee | 従業員とユニットの関連付けを解除する |
