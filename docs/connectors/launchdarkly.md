# LaunchDarkly コネクタ

公式: https://docs.workato.com/en/connectors/launchdarkly.html

## Triggers
| 名前 | 説明 |
|---|---|
| Updated feature flag targeting status | フィーチャーフラグのターゲティングステータスが更新されたときに検知するリアルタイムトリガー |
| New event | Feature Flags、Projects、Segments、Code References、Environmentsなど様々なオブジェクトを監視する汎用トリガー |

## Actions
| 名前 | 説明 |
|---|---|
| Create record | Feature FlagsやProjectsを作成する（既存フラグのクローンも可能） |
| Delete record | Feature Flags、Projects、Userオブジェクトを削除する |
| Get record details by ID | Feature Flags、Feature Flag Status、Flag Settings for Users、Projects、UsersのIDで詳細を取得する |
| List records | Feature Flag Status、Flag Settings for Users、Expiring User Targets on Feature Flagsを一覧表示する |
| Search records | Feature Flags、Projects、Usersを検索する |
| Switch feature flag status | フィーチャーフラグのオン/オフを切り替えるシンプルなアクション |
| Update record | Feature Flags、Feature Flag Settings for Users、Expiring User Targets、Feature Flag Targetsを更新する |
| Custom action | 特定のビジネスプロセスに合わせたカスタムアクションを作成する |
