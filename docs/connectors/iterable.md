# Iterable コネクタ

公式: https://docs.workato.com/en/connectors/iterable.html

## Triggers
| 名前 | 説明 |
|---|---|
| New object | システムWebhookに登録されたイベントをリッスンする |
| New workflow | 特定のIterableワークフローからWebhookイベントをキャプチャする |

## Actions
| 名前 | 説明 |
|---|---|
| Add subscribers to list | Iterableの指定リストにユーザーを追加する |
| Bulk upsert users | 複数ユーザーを一括で作成または更新する |
| Create list | Iterable内に新しいリストを作成する |
| Get user by email | メールアドレスでユーザー情報を取得する |
| List channels | Iterableで利用可能な通信チャネルを取得する |
| List project lists | Iterableプロジェクト内の全リストを表示する |
| Send SMS | Iterableを通じてSMSメッセージを送信する |
| Track event | ユーザーアクティビティに関連するカスタムイベントを記録する |
| Update user subscription | ユーザーのサブスクリプション設定を変更する |
| Upsert user | 新規ユーザーの作成または既存ユーザーデータの更新を行う |
