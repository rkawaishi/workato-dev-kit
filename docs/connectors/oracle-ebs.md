# Oracle E-Business Suite コネクタ

公式: https://docs.workato.com/en/connectors/oracle-ebs.html

## Triggers
| 名前 | 説明 |
|---|---|
| New business event | Oracle EBSで新しいビジネスイベントが発生した時にワークフローを開始する |
| New custom business event | カスタムビジネスイベントが作成された時にプロセスを開始する |

## Actions
| 名前 | 説明 |
|---|---|
| Execute PL/SQL operation | Oracle EBS内でPL/SQL操作を実行する |
