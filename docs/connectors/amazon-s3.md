# Amazon S3 コネクタ

公式: https://docs.workato.com/en/connectors/s3.html

## Triggers
| 名前 | 説明 |
|---|---|
| New CSV file | S3に新しいCSVファイルが作成されたときにトリガーされる |

## Actions
| 名前 | 説明 |
|---|---|
| Create bucket | 新しいS3バケットを作成する |
| Delete file/folder | S3からファイルまたはフォルダーを削除する |
| Download file contents | S3からファイルデータを取得する |
| Generate presigned URL | S3オブジェクトの署名付きURLを生成する |
| Get bucket location | バケットの地理的リージョンを取得する |
| List files in bucket | S3バケット内のすべてのファイルを一覧表示する |
| Upload file | S3にファイルをアップロードする |
| Upload file streaming | ストリーミングプロトコルを使用してS3にファイルを転送する |
| Use S3 as audit log streaming destination | 監査ログをS3ストレージに送信する |
