# Sage Intacct コネクタ

公式: https://docs.workato.com/en/connectors/intacct.html

## Triggers
| 名前 | 説明 |
|---|---|
| New AR payment | 新規作成された売掛金支払いを検知する |
| New contact | 新規作成された連絡先レコードを検知する |
| New expense | 新規作成された経費エントリを検知する |
| New invoice | 新規作成された請求書レコードを検知する |
| New item | 新規作成されたアイテムレコードを検知する |
| New project | 新規作成されたプロジェクトレコードを検知する |
| New project task | 新規作成されたプロジェクトタスクを検知する |
| New/updated AP bill | 買掛金請求書の作成または更新を監視する |
| New/updated AP payment | 買掛金支払いの変更を追跡する |
| New/updated contact | 連絡先レコードの更新を監視する |
| New/updated expense | 経費エントリの変更を追跡する |
| New/updated GL account | 総勘定元帳アカウントの変更を監視する |
| New/updated item | アイテムレコードの変更を追跡する |
| New/updated object | 汎用オブジェクトの作成または更新を監視する |
| New/updated project | プロジェクトレコードの変更を追跡する |
| New/updated project-task | プロジェクトタスクの変更を監視する |
| New/updated purchase order | 発注書の作成または更新を追跡する |
| New/updated vendor | ベンダーレコードの変更を監視する |

## Actions
| 名前 | 説明 |
|---|---|
| Create vendor | Intacctに新しいベンダーレコードを作成する |
| Update vendor | 既存のベンダー情報を更新する |
