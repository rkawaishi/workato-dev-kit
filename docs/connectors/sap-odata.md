# SAP OData コネクタ

公式: https://docs.workato.com/en/connectors/sap-odata.html

## Triggers
| 名前 | 説明 |
|---|---|
| New object | SAP ODataでオブジェクトが作成された時にトリガーされる |
| Updated object | SAP ODataで既存のオブジェクトが更新された時にトリガーされる |

## Actions
| 名前 | 説明 |
|---|---|
| Create object | SAP ODataサービスに新しいレコードを作成する |
| Update object | SAP ODataサービスの既存レコードを更新する |
| Search object | 指定した条件に一致するレコードを検索・取得する |
| Delete object | SAP ODataサービスからレコードを削除する |
| Upsert object | レコードの存在有無に基づいて作成または更新する |
| Get object details by ID | IDを指定して特定のレコードの完全な情報を取得する |
| Custom action | 標準CRUD以外のカスタム操作を実行する |
| Extract bulk data | SAP ODataから大量のデータを効率的にバッチ取得する |
| Batch request | 複数のCRUD操作を1つのリクエストで実行する |
