# SAP Concur コネクタ

公式: https://docs.workato.com/en/connectors/concur.html

## Triggers
| 名前 | 説明 |
|---|---|
| New expense report | 新しい経費レポートが作成された時に検知する |
| New expense report submission | 経費レポートが承認のために提出された時に発動する |
| New or updated expense report | 経費レポートが作成または変更された時に発動する |
| New or updated invoice | 請求書が作成または変更された時に発動する |
| New or updated user | ユーザーアカウントの作成またはプロフィール更新時にトリガーされる |

## Actions
| 名前 | 説明 |
|---|---|
| Create list item | 既存のリストに新しいアイテムを追加する |
| Create user | SAP Concurに新しいユーザーアカウントを作成する |
| Create users (batch) | 複数のユーザーアカウントを一括作成する |
| Create vendors (batch) | 複数のベンダーレコードを一括追加する |
| Delete list item | リストからアイテムを削除する |
| Get all attendee types (batch) | 全ての参加者タイプ分類を取得する |
| Get all expense group configurations (batch) | 全ての経費グループ設定詳細を取得する |
| Get all expense types (batch) | 全ての設定済み経費カテゴリを一覧表示する |
| Get all list item (batch) | 特定のリスト内の全アイテムを取得する |
| Get all lists | システム内の全リストを取得する |
| Get all payment types (batch) | 全ての支払タイプオプションを取得する |
| Get expense report details | 特定のレポートの詳細情報を取得する |
| Get invoice details | 請求書の完全な情報を取得する |
| Get itemizations of specific expense | 経費の明細詳細を取得する |
| Get user | ユーザープロフィール情報を取得する |
| Get user provisioning status details | ユーザープロビジョニングワークフローのステータスを取得する |
| Retrieve children of list item | 親リストアイテム配下の子アイテムを取得する |
| Search expense reports (batch) | 指定した条件に一致する経費レポートを検索する |
| Search users (batch) | 検索パラメータに基づいてユーザーアカウントを検索する |
| Search vendors (batch) | 検索条件に一致するベンダーレコードを検索する |
| Submit an expense report through a workflow | 経費レポートを承認ワークフローに送信する |
| Update user | 既存のユーザーアカウント情報を変更する |
| Update user (batch) | 複数のユーザーアカウントを一括更新する |
| Update vendors (batch) | 複数のベンダーレコードを一括変更する |
| Upload receipt image | 経費に領収書画像を添付する |
| Get entry image URL | 領収書画像のURLを取得する |
