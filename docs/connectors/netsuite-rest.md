# NetSuite REST コネクタ

公式: https://docs.workato.com/en/connectors/netsuite-rest.html

## Triggers

公式ドキュメント参照

## Actions
| 名前 | 説明 |
|---|---|
| Create record | NetSuiteに個別のレコードを作成する |
| Create records (async) | 複数レコードを非同期で一括作成する |
| Delete record | NetSuiteから単一のレコードを削除する |
| Delete records (async) | 複数レコードを非同期で一括削除する |
| Execute RESTlet | NetSuite内のカスタムRESTletスクリプトを実行する |
| Execute SuiteQL | SuiteQLクエリを実行してNetSuiteからデータを取得する |
| Get async job result | 非同期操作の結果を取得する |
| Get record by ID | NetSuite IDを使用して特定のレコードを取得する |
| Search records | 指定条件に一致するレコードを検索する |
| Update record | NetSuiteの既存レコードを1件更新する |
| Update records (async) | 複数レコードを非同期で一括更新する |
| Upsert record | 条件に基づいてレコードを作成または更新する |
| Upsert records (async) | 複数レコードを非同期で一括作成または更新する |
