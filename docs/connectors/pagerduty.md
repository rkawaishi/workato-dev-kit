# PagerDuty コネクタ

公式: https://docs.workato.com/en/connectors/pagerduty.html

## Triggers
| 名前 | 説明 |
|---|---|
| New incident | PagerDutyの新しいインシデントをポーリング間隔ごとにチェックする |
| New notification | PagerDutyの新しい通知をポーリング間隔ごとにチェックする |

## Actions
| 名前 | 説明 |
|---|---|
| Add note to incident | PagerDutyのインシデントにノートを追加する |
| Get incident by ID | インシデントIDで詳細を取得する |
| List log entries | 指定されたインシデントのログエントリを一覧表示する |
| Search incident | PagerDutyの既存インシデントを検索する |
| Send an event | トリガーイベントを送信して新しい問題を報告または進行中の問題を更新する |
| Update incident | 既存のインシデントを更新する |
