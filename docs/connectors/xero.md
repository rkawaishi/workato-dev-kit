# Xero コネクタ

公式: https://docs.workato.com/en/connectors/xero.html

## Triggers

| 名前 | 説明 |
|---|---|
| New/updated contact | Xeroで連絡先が作成または変更されたときに検出する |
| New/updated employee | 従業員レコードが作成または変更されたときに検出する |
| New/updated payment | 支払いが作成または変更されたときに検出する |
| New/updated invoice | 請求書が作成または変更されたときに検出する |

## Actions

| 名前 | 説明 |
|---|---|
| Search contacts | 名前、メールなどの条件で連絡先を検索する |
| Upsert contact | 新しい連絡先を作成するか、既存の連絡先を更新する |
| Create manual journal | 新しい手動仕訳を作成する |
| Search manual journals | 指定されたパラメータで手動仕訳を検索する |
| Update manual journal | 既存の手動仕訳を変更する |
| Create prepayment | 新しい前払いトランザクションを記録する |
| Search payments | 支払いレコードを検索する |
| Create employee | 新しい従業員レコードを追加する（US版） |
| Get payment | 特定の支払いの詳細を取得する |
| Create invoice payment | 請求書に対する支払いを記録する |
