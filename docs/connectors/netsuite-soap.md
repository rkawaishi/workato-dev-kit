# NetSuite SOAP コネクタ

公式: https://docs.workato.com/en/connectors/netsuite.html

## Triggers
| 名前 | 説明 |
|---|---|
| Export new and updated standard or custom records (bulk) | 新規作成および更新されたレコードを一括で取得する |
| New classification object | NetSuiteで新しい分類オブジェクトが作成されたときにトリガーされる |
| New standard object / New custom object | 新しい標準またはカスタムレコードが追加されたときにトリガーされる |
| Updated standard object / Updated custom object | 既存の標準またはカスタムレコードが変更されたときにトリガーされる |
| New standard object / New custom object (batch) | 新規作成されたレコードをバッチで処理する |
| New saved search result for object | 保存された検索が新しい結果を返したときにトリガーされる |
| New/updated saved search result for object | 保存された検索の新規および更新された結果でトリガーされる |
| Export new standard records | 新規作成された標準レコードをエクスポートする |
| Export new custom records | 新規作成されたカスタムレコードをエクスポートする |
| Export new/updated standard records | 新規作成および更新された標準レコードをエクスポートする |
| Export new/updated custom records | 新規作成および更新されたカスタムレコードをエクスポートする |
| Deleted standard record | 標準レコードが削除されたときにトリガーされる |

## Actions
| 名前 | 説明 |
|---|---|
| Create standard object / Create custom object | NetSuiteに新しい標準またはカスタムレコードを作成する |
| Create standard object / Create custom object (bulk) | 複数の標準またはカスタムレコードを一括作成する |
| Create standard record / Create custom record (batch) | バッチ処理でレコードを作成する |
| Get all standard objects | 指定タイプの全標準レコードを取得する |
| Get case comments | サポートケースに関連するコメントを取得する |
| Search standard objects | 標準レコードを検索・フィルタリングする |
| Search custom objects | カスタムレコードを検索・フィルタリングする |
| Update standard object / Update custom object | 既存の標準またはカスタムレコードを更新する |
| Update records (batch) | バッチ処理で複数レコードを更新する |
| Update records (bulk) | 一括操作で複数レコードを更新する |
| Upsert records (batch) | バッチ処理でレコードを作成または更新する |
| Upsert records (bulk) | 一括でレコードを作成または更新する |
| Delete standard record | 標準レコードを1件削除する |
| Delete standard records (batch) | バッチで複数の標準レコードを削除する |
| Delete custom record | カスタムレコードを1件削除する |
| Delete custom records (batch) | バッチで複数のカスタムレコードを削除する |
| Initialize record | デフォルト値で新しいレコードインスタンスを準備する |
