# Zoho CRM コネクタ

公式: https://docs.workato.com/en/connectors/zoho-crm.html

## Triggers

| 名前 | 説明 |
|---|---|
| New account | Zoho CRMで新しいアカウントが作成されたときにトリガーされる |
| New call | 新しい通話レコードが追加されたときにトリガーされる |
| New case | サポートケースが記録されたときにトリガーされる |
| New contact | 新しい連絡先が作成されたときにトリガーされる |
| New custom object | 新しいカスタムオブジェクトが作成されたときにトリガーされる |
| New deal | 新しい商談が作成されたときにトリガーされる |
| New invoice | 新しい請求書が生成されたときにトリガーされる |
| New standard object | 新しい標準オブジェクトレコードが作成されたときにトリガーされる |
| New vendor | 新しいベンダーが追加されたときにトリガーされる |
| New/updated account | アカウントの作成または変更時にトリガーされる |
| New/updated call | 通話レコードの新規作成または変更時にトリガーされる |
| New/updated case | ケースの作成または更新時にトリガーされる |
| New/updated contact | 連絡先の変更時にトリガーされる |
| New/updated contacts (batch) | 複数の連絡先の変更をバッチで処理する |
| New/updated custom object | カスタムオブジェクトの作成または更新時にトリガーされる |
| New/updated deal | 商談の作成または変更時にトリガーされる |
| New/updated lead | リードの作成または更新時にトリガーされる |
| New/updated leads (batch) | 複数のリードの変更をまとめて処理する |
| New/updated standard object | 標準オブジェクトの作成または変更時にトリガーされる |

| 名前 | 説明 |
|---|---|
| Batch create custom objects | 単一操作で複数のカスタムオブジェクトを作成する |
| Batch create standard objects | 複数の標準オブジェクトを一括作成する |
| Batch update custom objects | 複数のカスタムオブジェクトを一括更新する |
| Batch update standard objects | 複数の標準オブジェクトを一括更新する |
| Batch upsert custom objects | カスタムオブジェクトをバッチモードで作成または更新する |
| Batch upsert standard objects | 標準オブジェクトをバッチモードで作成または更新する |
| Bulk create custom objects | カスタムオブジェクトの大規模作成 |
| Bulk create standard objects | 標準オブジェクトの大規模作成 |
| Bulk update custom objects | カスタムオブジェクトの大規模更新 |
| Bulk update standard objects | 標準オブジェクトの大規模更新 |
| Bulk upsert custom objects | カスタムオブジェクトの大規模作成/更新 |
| Bulk upsert standard objects | 標準オブジェクトの大規模作成/更新 |
| Create account | 新しいアカウントレコードを追加する |
| Create campaign | 新しいキャンペーンを開始する |
| Create contact | 新しい連絡先レコードを作成する |
| Create deal | 新しい商談エントリを生成する |
| Create invoice | 新しい請求書ドキュメントを作成する |
| Create lead | 新しいリードを登録する |
| Create vendor | 新しいベンダーレコードを作成する |
| Delete custom object | カスタムオブジェクトを削除する |
| Delete standard object | 標準オブジェクトを削除する |
| Get lead by ID | IDを使用して特定のリードを取得する |

| Search contacts (batch) | 複数の連絡先をクエリする |

| Search invoice | 請求書レコードを検索する |

| Search products (batch) | 複数の製品をクエリする |
| Search records using COQL query | レコードに対してカスタムクエリを実行する |
| Search standard objects | 標準オブジェクトレコードをクエリする |
| Search vendors (batch) | 複数のベンダーをクエリする |
| Update account | 既存のアカウントを変更する |
| Update campaign | 既存のキャンペーンを変更する |
| Update contact | 既存の連絡先を変更する |

| Update lead | 既存のリードを変更する |
| Update vendor | 既存のベンダーを変更する |
| Upsert custom object | カスタムオブジェクトを作成または更新する |
| Upsert standard object | 標準オブジェクトを作成または更新する |
