# FTP/FTPS コネクタ

公式: https://docs.workato.com/en/connectors/ftp.html

## Triggers
| 名前 | 説明 |
|---|---|
| New CSV file in directory | FTP/FTPSディレクトリ内の新しいCSVファイルを監視する |

## Actions
| 名前 | 説明 |
|---|---|
| Download file | FTP/FTPSサーバーからファイルをダウンロードする |
| Upload file | FTP/FTPSサーバーにファイルをアップロードする |
| Rename file | サーバー上のファイル名を変更する |
| Remove file | サーバーからファイルを削除する |
