# SAP RFC コネクタ

公式: https://docs.workato.com/en/connectors/sap.html

## Triggers
| 名前 | 説明 |
|---|---|
| Receive outbound IDoc | SAPシステムからのアウトバウンドIDocを受信する |

## Actions
| 名前 | 説明 |
|---|---|
| Send inbound IDoc | SAPシステムにインバウンドIDocを送信する |
| Call remote function (RFC) | WorkatoからSAPのファンクションモジュールを呼び出す |
| Call BAPI | SAPのBAPIを呼び出す |
