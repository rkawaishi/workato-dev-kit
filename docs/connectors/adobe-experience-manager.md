# Adobe Experience Manager コネクタ

公式: https://docs.workato.com/en/connectors/adobe_experience_manager.html

## Triggers
| 名前 | 説明 |
|---|---|
| Asset added | AEMでアセットが追加されたときにトリガーされる |
| Asset updated | AEMでアセットが更新されたときにトリガーされる |
| Folder added | AEMでフォルダーが追加されたときにトリガーされる |

## Actions
| 名前 | 説明 |
|---|---|
| Search assets | 指定したプロパティフィルターと値に基づいてアセットを検索する |
| Update assets | 既存のアセットプロパティを変更、または新しいプロパティを作成する |
| Manage folders | フォルダー操作を管理する |
