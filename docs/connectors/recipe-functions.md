# Recipe function by Workato コネクタ

公式: https://docs.workato.com/en/connectors/recipe-functions.html

## Triggers
| 名前 | 説明 |
|---|---|
| New function call | 他のレシピがこのレシピファンクションを呼び出した時にトリガーされる |

## Actions
| 名前 | 説明 |
|---|---|
| Call recipe function asynchronously | レシピファンクションを非同期で呼び出す（完了を待たない） |
| Call recipe function synchronously | レシピファンクションを同期的に呼び出し、レスポンスを待つ |
| Return data from a recipe function | 呼び出し元のレシピに出力データを返す |
| Wait for async calls | 非同期レシピファンクション呼び出しの完了を待つ |
