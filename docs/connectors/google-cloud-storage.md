# Google Cloud Storage コネクタ

公式: https://docs.workato.com/en/connectors/google_cloud_storage.html

## Triggers
公式ドキュメント参照

## Actions
| 名前 | 説明 |
|---|---|
| Create bucket | Google Cloud Storageに新しいバケットを作成する |
| Delete bucket | 既存のバケットを削除する |
| Delete object | バケットから特定のオブジェクトを削除する |
| Download object | Cloud Storageに保存されたファイルを取得する |
| Get bucket | 特定のバケットの詳細情報を取得する |
| List buckets | プロジェクトに関連するすべてのバケットを一覧表示する |
| List objects | バケット内のファイルとオブジェクトを一覧表示する |
| Update bucket | 既存のバケットの設定を変更する |
| Update object metadata | 保存されたオブジェクトの属性とプロパティを変更する |
| Upload object with file streaming | ストリーム処理を使用してCloud Storageにファイルを転送する |
