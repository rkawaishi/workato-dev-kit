# Marketo コネクタ

公式: https://docs.workato.com/en/connectors/marketo.html

## Triggers
| 名前 | 説明 |
|---|---|
| New Marketo Self Service Flow Step | Marketoフローからデータを受信するトリガー |
| Export new leads (bulk) | 新規リードを一括エクスポートするトリガー |

## Actions
| 名前 | 説明 |
|---|---|
| Return data to Marketo Self Service Flow Step | リードを更新するためにMarketoにデータを返すアクション |
| Get leads | Marketoからリードフィールドを取得する |
| Update leads | Marketoのリードフィールドデータを更新する |
