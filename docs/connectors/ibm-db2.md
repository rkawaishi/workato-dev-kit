# IBM Db2 コネクタ

## Triggers
公式ドキュメント参照

## Actions
| 名前 | 説明 |
|---|---|
| Insert rows | Db2テーブルに新しいレコードを追加する |
| Upsert rows | 指定条件に基づいて新規レコードの挿入または既存レコードの更新を行う |
| Select rows | フィルタ条件を使用してDb2テーブルからレコードを取得する |
| Select rows with custom SQL | カスタムSQLを使用してレコードをクエリする |
| Delete rows | Db2テーブルからレコードを削除する |
| Run custom SQL | データベースに対して任意のSQLコマンドを実行する |
| Export query result | クエリ結果をエクスポートする |
