# Apache Kafka コネクタ

公式: https://docs.workato.com/en/connectors/kafka.html

## Triggers
| 名前 | 説明 |
|---|---|
| New message | Kafkaトピックから個別のメッセージを受信する |
| New messages (batch) | Kafkaトピックからバッチでメッセージを受信する |

## Actions
| 名前 | 説明 |
|---|---|
| Publish message | Kafkaトピックに単一のメッセージを送信する |
| Publish messages (batch) | Kafkaトピックにバッチでメッセージを送信する |
