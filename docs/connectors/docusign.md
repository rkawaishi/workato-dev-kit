# DocuSign コネクタ

公式: https://docs.workato.com/en/connectors/docusign.html

## Triggers

| 名前 | 説明 |
|---|---|
| New document event | DocuSignで新しいドキュメントイベントが発生したときにトリガーする |

## Actions

| 名前 | 説明 |
|---|---|
| Create/send document | 署名用のドキュメントを作成して送信する |
| Send document using a template | 定義済みのDocuSignテンプレートを使用してドキュメントを送信する |

> 上記は公式ドキュメントのトップページで確認できたもののみです。追加のトリガー・アクションについては公式ドキュメント参照。
