# Box コネクタ

公式: https://docs.workato.com/en/connectors/box.html

## Triggers

| 名前 | 説明 |
|---|---|
| New/updated file in folder | Boxフォルダ内でファイルが新規作成または更新されたときにトリガーする |

## Actions

| 名前 | 説明 |
|---|---|
| Upload file content | プレーンテキストファイルのコンテンツを指定のBoxフォルダにアップロードする |
| Upload file from URL | URLからファイルを直接Boxにアップロードする |
| Create shared link | Boxファイルまたはフォルダへの共有リンクを生成する |

> 上記は公式ドキュメントのトップページで確認できたもののみです。追加のトリガー・アクションについては公式ドキュメント参照。
