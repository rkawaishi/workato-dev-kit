## Triggers
| 名前 | 説明 |
|---|---|
| New account | 新しいアカウントが作成された時にトリガーする |
| New bank deposit | 新しい銀行入金が作成された時にトリガーする |
| New bill payment | 新しい支払いが作成された時にトリガーする |
| New credit note | 新しいクレジットノートが作成された時にトリガーする |

| New employee | 新しい従業員が作成された時にトリガーする |
| New estimate | 新しい見積りが作成された時にトリガーする |
| New invoice | 新しい請求書が作成された時にトリガーする |
| New item | 新しいアイテムが作成された時にトリガーする |
| New payment | 新しい支払いが作成された時にトリガーする |
| New sales receipt | 新しい売上レシートが作成された時にトリガーする |
| New vendor | 新しいベンダーが作成された時にトリガーする |
| New/updated account | アカウントの新規作成または更新時にトリガーする |
| New/updated tax code | 税コードの新規作成または更新時にトリガーする |
| New/updated tax rate | 税率の新規作成または更新時にトリガーする |
| Updated bill | 既存の請求書が変更された時にトリガーする |
| Updated credit note | 既存のクレジットノートが変更された時にトリガーする |
| Updated customer | 既存の顧客が変更された時にトリガーする |
| Updated employee | 既存の従業員が変更された時にトリガーする |
| Updated estimate | 既存の見積りが変更された時にトリガーする |
| Updated invoice | 既存の請求書が変更された時にトリガーする |
| Updated item | 既存のアイテムが変更された時にトリガーする |
| Updated purchase | 既存の購入が変更された時にトリガーする |
| Updated vendor | 既存のベンダーが変更された時にトリガーする |
| Deleted item | アイテムが削除された時にトリガーする |
