## Triggers

公式ドキュメント参照

## Actions
| 名前 | 説明 |
|---|---|
| Delete documents | MongoDB Atlasコレクションからドキュメントを削除する |
| Insert documents | MongoDB Atlasコレクションに新しいドキュメントを追加する |
| Replicate documents | MongoDB Atlas内でドキュメントを複製する |
| Search documents | コレクションからドキュメントを検索・取得する |
| Update documents | MongoDB Atlasコレクション内の既存ドキュメントを更新する |
