# RecipeOps by Workato コネクタ

公式: https://docs.workato.com/en/connectors/recipeops.html

## Triggers
| 名前 | 説明 |
|---|---|
| Account connected | ワークスペースアカウントが接続された時に検知する |
| Account credentials refresh failed | アカウント資格情報の更新が失敗した時にアラートする |
| Account disconnected | ワークスペースアカウントが切断された時にトリガーされる |
| API concurrency threshold exceeded | API同時実行数の閾値を超えた時に発動する |
| API policy quota violation | APIクォータポリシーに違反した時に発動する |
| API policy rate limit violation | APIレート制限ポリシーに違反した時に発動する |
| Deployment approved | デプロイメントが承認された時にトリガーされる |
| Deployment complete | デプロイメントが完了した時に発動する |
| Deployment failed | デプロイメントが失敗した時に発動する |
| Deployment rejected | デプロイメントが拒否された時に発動する |
| Deployment re-opened for review | デプロイメントがレビュー状態に戻された時にトリガーされる |
| Job failed | ジョブの実行が失敗した時に検知する |
| Member invitation accepted | ワークスペースメンバーが招待を承諾した時に発動する |
| New deployment submitted for review | デプロイメントが承認待ちになった時にトリガーされる |
| On-prem agent disconnected | オンプレミスエージェントが切断された時に発動する |
| Package deployed | パッケージがデプロイされた時に発動する |
| Recipe started | レシピの実行が開始された時に検知する |
| Recipe stopped by user | ユーザーがレシピを手動で停止した時にトリガーされる |
| Recipe stopped by Workato | プラットフォームがレシピを停止した時に発動する |
| Usage threshold reached | アカウント使用量が定義された閾値に達した時にアラートする |

## Actions
| 名前 | 説明 |
|---|---|
| Get account details | ワークスペースアカウント情報を取得する |
| Get recipe details | 特定のレシピの設定とメタデータを取得する |
| List connections | ワークスペースの全接続を一覧表示する |
| List recipes | ワークスペースの全レシピを一覧表示する |
| Rerun jobs | 以前に失敗または完了したジョブを再実行する |
| Search job history | ジョブ実行ログと履歴を検索する |
| Search recipes | 指定した条件に一致するレシピを検索する |
| Start recipe | レシピの実行を開始する |
| Stop recipe | 実行中のレシピを停止する |
