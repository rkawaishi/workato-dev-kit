# Workfront コネクタ

公式: https://docs.workato.com/en/connectors/workfront.html

## Triggers

| 名前 | 説明 |
|---|---|
| New event | Workfrontで新しいイベントをリアルタイムで検出し、各イベントごとに個別のジョブを作成する |
| New object | 設定された間隔でWorkfrontの新規作成オブジェクトをポーリングする |
| New or updated object | 定期的なポーリングで新規作成および変更されたオブジェクトの両方を監視する |

## Actions

| 名前 | 説明 |
|---|---|
| Create object | Workfrontにオブジェクトを作成する（オプションでカスタムフォームを添付可能） |
| Update object | Workfrontのオブジェクトを更新する（カスタムフォームの変更・添付を含む） |
| Search objects | 入力フィールドまたはWorkfrontフィルタ構文を使用して任意のオブジェクトを検索する |
