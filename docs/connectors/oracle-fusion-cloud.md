# Oracle Fusion Cloud コネクタ

公式: https://docs.workato.com/en/connectors/oracle-fusion-cloud.html

## Triggers
| 名前 | 説明 |
|---|---|
| New business event | Oracle Fusion Cloudからリアルタイムでビジネスイベントを取得する |
| New employee atom feed entry | Atomフィード経由で新しい従業員レコードを検出する |
| New organization atom feed entry | Atomフィードで新しい組織エントリを監視する |
| New record | 新しいレコードが作成された時にトリガーする |
| New record (batch) | 新しく作成された複数のレコードをバッチで処理する |
| New/updated record | 新規作成または変更されたレコードに応答する |
| New/updated record (batch) | 新規または変更されたレコードをバッチで処理する |

## Actions
| 名前 | 説明 |
|---|---|
| Append file comment | ファイルにコメントを追加する |
| Confirm extract consumption | データ抽出の消費を確認する |
| Create record | 新しいレコードを追加する |
| Create records (batch) | 複数のレコードを一括で追加する |
| Create user | 新しいユーザーアカウントを作成する |
| Delete record | レコードを削除する |
| Download ESS job execution details | 完了したジョブの詳細を取得する |
| Download export output | エクスポートされたデータファイルを取得する |
| Export bulk data | 大量データをシステムからエクスポートする |
| Extract and purge | 抽出後にデータを削除する |
| Fetch extract output | 抽出データの結果を取得する |
| Get flow task instance status | ワークフロータスクの進捗を確認する |
| Get record | 特定のレコード情報を取得する |
| Import bulk data | 大量データをシステムにインポートする |
| List entities | 利用可能なデータエンティティを表示する |
| Load and import data | データを取り込んで処理する |
| Search documents by file prefix | ファイル名パターンでドキュメントを検索する |
| Search records | レコードをバッチモードで検索する |
| Search records using advanced query (batch) | 複数レコードに対する高度な検索を実行する |
| Submit and get flow instance ID | ワークフローを開始しIDを取得する |
| Submit ESS job request | エンタープライズスケジューラジョブをキューに追加する |
| Submit job with output | ジョブを実行し結果を取得する |
| Update interface data | システムインターフェースのデータを変更する |
| Update record | 既存レコードを更新する |
| Update records (batch) | 複数のレコードを一括更新する |
| Upload to UCM | Universal Content Managementにファイルを保存する |
| Upsert record | レコードの作成または更新を行う |
