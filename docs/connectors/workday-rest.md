# Workday REST コネクタ

公式: https://docs.workato.com/en/connectors/workday-rest.html

## Triggers

| 名前 | 説明 |
|---|---|
| New worker | Workdayで新しいワーカーが作成されたときにワークフローを開始する |

## Actions

| 名前 | 説明 |
|---|---|
| Create a job change | 従業員のジョブ変更レコードを作成する |
| Get inbox tasks | Workdayの受信ボックスから保留中のタスクを取得する |
| Approve/reject inbox task | 受信ボックスタスクの承認または拒否を処理する |
| Get worker by ID | ワーカーIDを使用して特定の従業員情報を取得する |
| Search workers | 検索条件に基づいてワーカーを検索する |
