# Replicon コネクタ

公式: https://docs.workato.com/en/connectors/replicon.html

## Triggers
| 名前 | 説明 |
|---|---|
| New client | Repliconで新しいクライアントが作成された時に検知する |
| New project | Repliconで新しいプロジェクトが作成された時に検知する |
| New user | Repliconで新しいユーザーが作成された時に検知する |
| New/updated ready to sync invoice | 同期可能な請求書が新規作成または更新された時に検知する |
| Updated timesheet | タイムシートが変更された時に検知する |
