# Azure OpenAI コネクタ

公式: https://docs.workato.com/en/connectors/azure-openai.html

## Triggers

| 名前 | 説明 |
|---|---|
| (なし) | このコネクタにはトリガーがありません |

## Actions

| 名前 | 説明 |
|---|---|
| Complete text prompt | プロンプトからテキスト補完を生成する |
| Generate images | テキスト説明に基づいて画像を生成する |
| Generate text embedding | テキスト入力からベクトル埋め込みを生成する |
| Send messages to ChatGPT | Azure OpenAI経由でChatGPTモデルと会話する |
