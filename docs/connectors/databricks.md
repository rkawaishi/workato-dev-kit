# Databricks コネクタ

公式: https://docs.workato.com/en/connectors/databricks.html

## Triggers

| 名前 | 説明 |
|---|---|
| New rows using custom SQL (batch) | カスタムSQLクエリを使用してDatabricksからバッチでデータを取得する |
| New rows (batch) | Databricksテーブルから新規作成された行をバッチで取得する |
| New/updated rows (batch) | Databricksの新規および更新された行をバッチで監視・取得する |
