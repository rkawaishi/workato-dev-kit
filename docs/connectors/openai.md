# OpenAI コネクタ

公式: https://docs.workato.com/en/connectors/openai.html

## Triggers
なし

## Actions
| 名前 | 説明 |
|---|---|
| Business actions | 一般的なOpenAIユースケース向けの事前構築ワークフロー |
| Send messages to OpenAI models | ChatGPTなどのモデルと会話形式でやり取りする |
| Moderate text | コンテンツのポリシー準拠を評価する |
| Generate text embedding | テキストをベクトル表現に変換する |
| Generate images | OpenAIの画像生成機能を使って画像を作成する |
| Transcribe recording | 音声ファイルをテキストに変換する |
| Translate recording | 音声コンテンツを言語間で変換する |
