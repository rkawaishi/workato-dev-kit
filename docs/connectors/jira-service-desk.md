# Jira Service Desk コネクタ

公式: https://docs.workato.com/en/connectors/jsd.html

## Triggers
公式ドキュメント参照

## Actions
| 名前 | 説明 |
|---|---|
| Create customer | Jira Service Deskに新しいカスタマーレコードを作成する |
| Create customer request | カスタマーからの新しいサポートリクエストを生成する |
| Create comment | 既存のイシューまたはリクエストにコメントを追加する |
| List comments | 特定のリクエストに関連する全コメントを取得する |
| Get comment by ID | 一意の識別子を使用して単一のコメントを取得する |
| Get queues | サービスデスクで利用可能なサポートキューを取得する |
| Get issues in queue | 指定したキュー内の全イシューを一覧表示する |
